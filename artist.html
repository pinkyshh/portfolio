Design & Solution

  • Requirements
  • Design

- The website lists at least 5 product items. Create each page for each product.
- Design a way to display the product description when the user clicks on the product item.
- Add a Payment Calculator as shown in the image.



- Create interest rate as a toggle-button group. The group contains 3 buttons: 0%, 4%, & 8%.

My goal was to make the website look elegant while also incorporating all of the criteria. The theme was chosen to complement the colors of the art works and to appeal to the user. I made a collection of 3 product categories with 2 items in each.



The objective of this calculator, which was developed using JavaScript, is to allow the user to estimate the cost of purchasing an art piece. For the calcultor, I decided to use a slider for the amount making it easier for the user to input their value instead of typing it in. Based on the requirement, I used a toggle button for the interest rate and did the same for the duration.
